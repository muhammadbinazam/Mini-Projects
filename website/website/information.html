Web Technology Information

What is HTML?

HTML stands for HyperText Markup Language. It is the core language used to create the structure and content of a web page. Every heading, paragraph, image, and link you see on a website is defined by HTML tags. Think of HTML as the skeleton of the website.

What is CSS?

CSSstands for Cascading Style Sheets. It is used to control the visual appearance, or style, of the content created with HTML. This includes colors, fonts, layout, spacing, and animations. CSS is what makes a website look good; it's the clothes and makeup on the skeleton!
